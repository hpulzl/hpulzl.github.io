Linux基本指令学习（二）

发表于 2016-10-15   |   分类于 Linux学习   |     |   阅读次数

- 1.VIM编辑器

VIM –编辑器

  • 编辑today.txt文件 vim today.txt
  • 数据命令i 进入插入模式
  • 输入内容 AAABBBCCC
  • ctrl+c 退出插入模式或者敲击ESC切换命令
  • :wq +回车 –保存并退出
  • :q! +回车 –强制退出
  • ：set number + 回车 –设置行号
  • /+字符串 查找关键字
1 nihao!
2 AAABBBCCC
3 helerj
4 shenmegui
:set number

-2.用户和组管理

添加一个账户

useradd -m zyg 参数-m用来设定系统添加账户时自动添加一个根目录。

root@ubuntu:/home/lzl/lzl/linuxStudy# useradd -m zyg

修改zyg账户的登录名称

root@ubuntu:/home/lzl/lzl/linuxStudy# usermod -l zhuyg zyg

锁定用户账号密码

usermod -L zhuyg 锁定账号不可使用

解锁用户账号密码

usermod -U zhuyg

添加一个分组

root@ubuntu:/home/lzl# groupadd superman

修改superman分组

root@ubuntu:/home/lzl# groupmod -g 355 superman

删除superman分组

root@ubuntu:/home/lzl# groupdel superman

切换用户

  • su -用户名 –切换用户
  • sudo -i 切换到root

删除zyg账号 

userdel -r zyg (-r 连同用户目录一起删除)

查看分组

cat /etc/group

-3.文件权限管理

  • 以管理员hhhh进行文件操作，会提示
hhhh@ubuntu:/home$ cd /home/lzl/lzl/familyA/
hhhh@ubuntu:/home/lzl/lzl/familyA$ vim son.txt 
E45: 'readonly' option is set (add ! to override)

修改son.txt文件的权限

root@ubuntu:/home/lzl/lzl/familyA# chmod o+w son.txt
