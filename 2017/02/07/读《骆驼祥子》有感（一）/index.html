读《骆驼祥子》有感（一）

发表于 2017-02-07   |   分类于 读书笔记   |     |   阅读次数

有些书只有到一定年纪才能读懂那些道理。

        “骆驼”祥子，这个修饰词不是因为他的外形驼背，而是他当俘虏逃出来时候顺手牵了三只骆驼而得名！骆驼最后卖了30块钱，还不及他之前买的那辆拉车钱的一半。

        是的，祥子就是靠拉车，卖力气过活。那时候叫车夫。拉车这一行业是一个靠体力干的活。卖力气一天来回大街小巷跑，衣服湿了又干，可是也挣不了钱。祥子无父无母，没有亲戚和朋友。十几岁就出来打拼，现在已经二十多岁，他看到同行人，年龄四五十的依旧是拉着破车，整天衣不蔽体，吃了上顿没下顿。甚至有饿死或冻死的可能。可是，社会就是那样，没有显赫的家世，尊贵的地位，没有基础家业，祥子只能靠自己，一步一步往上爬，一步一个脚印的挣铜子！好在，祥子的想法很端正，他要靠自己的双脚去闯一片天，讨一个老婆，营造一个温馨美满的未来。他是这样想的，他努力的攒钱，一年两年下来，生病忍着不看病，也不去花天酒地的场所，不和其他车夫聊闲话，不喝酒不抽烟。他攒下来100块钱，买了一辆崭新的包车。他要去富贵人家拉包月车，这样每月除了基本工资，还有一些打赏，这样他攒足了钱，就能再买辆车 他就可以当租车的人。慢慢的成为车场老板，再找个漂亮的媳妇，他的日子就红火了，再不像那些劳苦的车夫一样，到老了还是在雪地里拉着破烂车！

        可是，日子都不是一番风顺。而祥子像是上天特意选中的灾难者一样。他的新车还没有为他盈利，就被抓去当了俘虏。车被没收，人也搭进去了。老天没有让他一死了之，是因为还有更多的灾难让他去承受…

    经过了这一次变动，他几年的努力算是白费了，好在他对生活还是有些信心，他要再次振作起来。此时的他，还拒绝攀岩富贵（拒绝和丑陋的车场老板刘四爷的女儿虎妞成亲），他还兢兢业业的干着老本行，并且在一个很不错的富贵人家（孙先生）拉包月车。可是好景不长，孙先生被人报复揭发说他是革命份子。祥子也因此受到牵连，自己辛苦攒的钱，再次落空！并且接近年关（全国人民欢聚，幸福的时刻），又是冬天北平极冷的时候。他躲在隔壁老李家，外面下着大雪，他头疼得抬不起来，可是怎么也睡不着。祥子再一次尝到了绝望的滋味。生活真的很艰难！

        有些故事，到了某些年纪我们才能读懂！虽不在同一时代，但是相仿的年龄。现在的我，生活物质上比祥子要好上千万倍。这是我知足的地方！但在当下的社会环境，又到了男大当婚的年龄，然而抛开感情不论，在结婚这一物质层面，一套房子面前，我渺小的就像一只蚂蚁。不光我，应该是当下大部分的人！可是，越是这样艰苦，越要有坚定的信念，要相信爱情会来，要相信房子会有，要相信未来十分光明。

        知道祥子最终结局很凄惨，但是要明白，他为什么会是那样的结局。弯路可能不能避免，但是不能一错再错！
